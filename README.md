## Platform

Currently runs with:

- .NET Core 2.1.0-rc1-final
- SignalR (.NET Core) 1.0.0-rc1-final w/ Redis caching
- Angular v6.0.3
- Angular-CLI v6.0.3
- Electron v2.0.1
- Electron Builder v20.13.4
